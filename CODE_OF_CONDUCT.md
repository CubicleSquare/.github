# Cubicle Square Code of Conduct

We aim to maintain a productive, inclusive, and respectful engineering culture.

---

## Expected Behavior

- Treat all team members with respect  
- Provide constructive, actionable feedback  
- Value diversity of thought  
- Follow engineering and security best practices  

---

## Unacceptable Behavior

- Harassment, discrimination, or abusive language  
- Publishing private or confidential information  
- Knowingly introducing vulnerabilities  
- Sabotaging work or PR reviews
